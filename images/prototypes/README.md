## ðŸ”§ Technical Context

All prototypes shown here are based on the ESP32-S3 platform and were validated at Clinique Hartmann (Paris) during a 3-month clinical trial involving 50+ patients.

**Specifications:**
- Autonomy: 8-10 hours continuous
- Sensors: MAX86916 PPG (4 channels), LIS3DH accelerometer
- Connectivity: WiFi + BLE
- Storage: SD card + AWS cloud
